# Virtual Box

## Imp shortcuts

- `(right-control)+f` to switch between full screen mode
- `(right-control)+g`

## To go to full screen

```
First know what your monitor resolution is
```

- Start the vm
- Go to `view` > enable `scalled mode`
- change vm OS display settings and set it to display settings of your host OS.
- if this doesn't work then try other options from the `view` menu.
